# svyGMaps

Google Maps Component for use with Servoy's NGClient

Getting started
-------------

Get the latest version from the [releases](https://github.com/Servoy/svyGMaps/releases). 
Download and import the googleMapsExample.servoy example solution with the matching version of the package from [releases](https://github.com/Servoy/svyGMaps/releases) or check out source directly from github.
